## Personal details that must never be published

Regardless of source (including the previous Firebase Studio portfolio — see `docs/PREVIOUS_PORTFOLIO_REFERENCE_AUDIT.md`), never publish:

- home address, telephone number, or date of birth;
- passport, visa, or immigration-status details;
- student numbers, salary, or other financial details;
- private references or third-party personal data;
- Firebase credentials, service-account keys, API keys, `.idx` configuration, or any other Firebase workspace secret;
- Vercel deployment tokens, environment variable values, or project secrets;
- confidential FIS client names, application names, or unsupported management/leadership/monetary claims.
